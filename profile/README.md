XDeNovo (纽肽生物) builds an AI-native biotechnology platform for de novo peptide and protein design. We connect computational biology, structure prediction, sequence generation, docking, scoring, and design-test-optimize loops into practical systems for discovering peptide binders and therapeutic candidates.

Our work spans both molecule design and the infrastructure needed to run it: internal MCP services for long-running GPU workflows, artifact tracking, and reproducible model operations, plus agentic tools that make complex bio-design workflows easier to compose and operate.

## What We Build

| Area | What it means |
| --- | --- |
| **AI peptide design** | Therapeutic peptides and antibodies designed for high affinity, high specificity, and low immunogenicity. |
| **Targeting heads** | Peptide targeting heads for PDC/PRC programs and other precision delivery modalities. |
| **Protein engineering** | Antimicrobial peptides, cosmetic peptides, and custom functional proteins. |
| **Bio-design infrastructure** | MCP services for queued compute jobs, GPU worker orchestration, artifact retrieval, and reproducible internal workflows. |
| **Agentic R&D tools** | Low-code multi-agent orchestration for composing agents, tools, knowledge, memory, and scientific workflows. |

## Featured Systems

Selected repositories include internal AI workflow services and agent tooling. Access may depend on GitHub organization permissions.

| Repository | Focus | Stack |
| --- | --- | --- |
| [**AgnoFlow**](https://github.com/XDenovo/AgnoFlow) | Local-first, low-code multi-agent platform for visually composing agents, tools, knowledge, memory, teams, and workflows with bring-your-own-key provider credentials. | React Flow, FastAPI, Agno |
| [**bindcraft-mcp**](https://github.com/XDenovo/bindcraft-mcp) | FastMCP service for BindCraft binder-design workflows with queued GPU jobs, artifact tracking, and ranked design retrieval. | Python, FastMCP, SQLite |
| [**pepmimic-mcp**](https://github.com/XDenovo/pepmimic-mcp) | FastMCP service for PepMimic peptide generation, scoring, relaxation, and candidate selection workflows. | Python, FastMCP, SQLite |
| [**graphpep-mcp**](https://github.com/XDenovo/graphpep-mcp) | FastMCP service for GraphPep protein-peptide complex scoring with receptor/decoy uploads and `scores.csv` artifact retrieval. | Python, FastMCP, SQLite |

## Platform Direction

We are building toward a unified AI bio-design stack:

- Generate and optimize peptides against challenging targets.
- Score, rank, and select candidates through reproducible computational pipelines.
- Run long-lived GPU and CPU jobs behind authenticated MCP interfaces.
- Preserve run metadata, logs, and artifacts for downstream analysis.
- Compose scientific workflows with low-code multi-agent tooling.

## Collaborate

We are interested in research and translational collaborations across AI protein design, peptide therapeutics, difficult-to-drug targets, computational biology infrastructure, wet-lab validation, and preclinical translation.
